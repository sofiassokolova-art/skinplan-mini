# Импорт продуктов из CSV

## Использование

```bash
# Импорт из файла по умолчанию (scripts/goldapple-import.csv)
npx tsx scripts/import-products-from-csv.ts

# Импорт из указанного файла
npx tsx scripts/import-products-from-csv.ts /path/to/file.csv
```

## Формат CSV

CSV файл должен содержать следующие колонки:

- **Название** - название продукта (обязательно)
- **Бренд** - название бренда (обязательно)
- **Ссылка** - ссылка на Gold Apple (обязательно)
- **Шаг** - stepCategory (moisturizer_rich, mask_sleeping и т.д.) (обязательно)
- **Категория** - базовый шаг (moisturizer, mask, spf и т.д.) (обязательно)
- **Типы кожи** - через запятую (dry, oily, combo, sensitive и т.д.)
- **Проблемы** - через запятую (acne, pigmentation, redness и т.д.)
- **Активные ингредиенты** - через запятую
- **Избегать при** - через запятую
- **Hero** - "Да" или "Нет"
- **Приоритет** - число (0 по умолчанию)
- **Опубликован** - "Да" или "Нет"
- **Цена** - число (опционально)
- **Объем** - строка (опционально)
- **Описание** - текст (опционально)
- **Описание для пользователя** - текст (опционально)
- **Состав** - текст (опционально)

## Что делает скрипт

1. Читает CSV файл
2. Парсит данные из CSV
3. Создает/обновляет бренды в БД (по названию)
4. Генерирует slug для каждого продукта
5. Создает/обновляет продукты в БД
6. Выводит статистику импорта

## Примечания

- Если продукт с таким же названием и брендом уже существует, он будет обновлен
- Бренды создаются автоматически, если их нет в БД
- Slug генерируется автоматически из названия бренда и продукта
