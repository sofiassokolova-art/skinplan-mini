# JWT_SECRET для Vercel

## Что такое JWT_SECRET?

`JWT_SECRET` - это секретный ключ, который используется для подписи и проверки JWT токенов авторизации. Он используется для:
- Авторизации пользователей через Telegram
- Авторизации админов в админ-панели
- Безопасного хранения сессий

## ⚠️ ВАЖНО

- **НИКОГДА не коммитьте JWT_SECRET в Git!**
- Используйте **разные ключи** для разработки и продакшена
- Храните ключ в переменных окружения Vercel

## Как добавить в Vercel:

1. Откройте Vercel Dashboard → Ваш проект → Settings → Environment Variables

2. Добавьте новую переменную:
   - **Name**: `JWT_SECRET`
   - **Value**: (скопируйте ключ ниже)
   - **Environment**: `Production`, `Preview`, `Development` (все три)

3. После добавления перезапустите деплой или подождите следующего коммита

**Скопируйте этот ключ полностью и вставьте в Vercel как значение переменной `JWT_SECRET`**

---

## Рекомендации:

- Длина: минимум 64 символа (128 в hex)
- Храните в безопасном месте (менеджер паролей)
- Не используйте один и тот же ключ для разных проектов
- Периодически обновляйте ключ (но это потребует повторной авторизации всех пользователей)
