# Шрифты для анкеты

Эта папка содержит локальные файлы шрифтов для анкеты.

## Необходимые файлы:

### Unbounded:
- `unbounded-regular.woff2` (weight: 400)
- `unbounded-bold.woff2` (weight: 700)

### Inter:
- `inter-thin.woff2` (weight: 100)
- `inter-extralight.woff2` (weight: 200)
- `inter-light.woff2` (weight: 300)
- `inter-regular.woff2` (weight: 400)
- `inter-medium.woff2` (weight: 500)
- `inter-semibold.woff2` (weight: 600)
- `inter-bold.woff2` (weight: 700)
- `inter-extrabold.woff2` (weight: 800)
- `inter-black.woff2` (weight: 900)

## Где скачать:

1. **Unbounded**: https://fonts.google.com/specimen/Unbounded
   - Нажмите "Download family"
   - Распакуйте архив
   - Конвертируйте .ttf файлы в .woff2 (можно использовать https://cloudconvert.com/ttf-to-woff2)
   - Переименуйте файлы согласно списку выше

2. **Inter**: https://fonts.google.com/specimen/Inter
   - Нажмите "Download family"
   - Распакуйте архив
   - Конвертируйте .ttf файлы в .woff2
   - Переименуйте файлы согласно списку выше

## Альтернативный способ:

Можно использовать онлайн-конвертер для скачивания и конвертации:
- https://gwfh.mranftl.com/fonts/unbounded
- https://gwfh.mranftl.com/fonts/inter

Эти сервисы позволяют скачать шрифты уже в формате .woff2.
