# Настройка домена для Telegram Login Widget

## Проблема: "Bot domain invalid"

Эта ошибка возникает, когда домен не добавлен в настройки бота в BotFather.

## Решение:

1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду: `/mybots`
3. Выберите вашего бота
4. Выберите "Bot Settings"
5. Выберите "Domain"
6. Введите домен вашего приложения (например: `skinplan-mini.vercel.app`)
7. Telegram проверит домен и сохранит его

## Важно:

- Домен должен точно совпадать с доменом, на котором работает приложение
- Для Vercel это обычно: `your-project.vercel.app`
- После добавления домена может потребоваться несколько минут для активации

## Проверка:

После добавления домена, перезагрузите страницу админки. Кнопка Telegram Login Widget должна появиться без ошибки "Bot domain invalid".
