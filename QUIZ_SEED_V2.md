# Seed Questionnaire v2

Скрипт для заполнения базы данных вопросами анкеты согласно QUIZ_FLOW.md.

## Использование

```bash
npm run seed:questionnaire-v2
```

## Что делает скрипт

1. Создает новую анкету с версией 2
2. Создает все группы вопросов согласно QUIZ_FLOW.md:
   - Цели ухода (1 вопрос)
   - Общая информация (2 вопроса)
   - Особенности кожи (4 вопроса)
   - Данные о здоровье (4 вопроса)
   - Текущий уход (4 вопроса)
   - Привычки и образ жизни (4 вопроса)
   - Предпочтения в уходе (3 вопроса)
3. Деактивирует все старые версии анкет
4. Активирует новую версию 2

## Коды вопросов

Все коды вопросов соответствуют тем, что используются в `app/(miniapp)/quiz/info-screens.ts` через `showAfterQuestionCode`:

- `skin_goals` - Какие ваши основные цели для кожи?
- `age` - Возраст
- `gender` - Пол
- `skin_type` - Тип кожи
- `skin_concerns` - Что вас больше всего беспокоит?
- `skin_sensitivity` - Уровень чувствительности
- `seasonal_changes` - Меняется ли состояние кожи в зависимости от сезона?
- `medical_diagnoses` - Есть ли у вас диагнозы?
- `pregnancy_breastfeeding` - Беременность/кормление (только для женщин)
- `allergies` - Аллергические реакции
- `avoid_ingredients` - Исключить ингредиенты
- `retinoid_usage` - Использовали ли ретинол?
- `retinoid_reaction` - Как кожа реагировала?
- `prescription_topical` - Рецептурные кремы/гели
- `oral_medications` - Пероральные препараты
- `makeup_frequency` - Частота использования декоративной косметики
- `spf_frequency` - Частота использования SPF
- `sun_exposure` - Время на солнце
- `lifestyle_habits` - Привычки образа жизни
- `care_type` - Тип ухода
- `care_steps` - Количество шагов
- `budget` - Бюджет

## Важно

- Скрипт удаляет старую анкету v2, если она существует
- Все старые активные анкеты автоматически деактивируются
- Коды вопросов должны точно совпадать с теми, что используются в `info-screens.ts`
