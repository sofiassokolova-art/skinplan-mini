# Настройка Telegram Login Widget

## Проблема: Сообщение подтверждения не приходит

Telegram Login Widget требует настройки домена в BotFather перед использованием.

## Решение: Настроить домен в BotFather

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/setdomain`
3. Выберите вашего бота из списка (например, `@skinplanned_bot`)
4. Введите домен вашего сайта: `skinplan-mini.vercel.app`

**Важно:**
- Домен должен быть без `https://` и без `/` в конце
- Telegram требует HTTPS для работы виджета
- После настройки домена виджет начнет работать корректно

## Альтернативное решение: Использовать Mini App авторизацию

Если Telegram Login Widget не работает, можно использовать авторизацию через Telegram Mini App:

1. Откройте админ-панель через Mini App (не через обычный браузер)
2. Если вы открыли через Mini App, авторизация произойдет автоматически через `initData`

## Отладка

После настройки домена:

1. Откройте страницу `/admin/login` в браузере
2. Введите номер телефона в виджете
3. Подождите сообщение подтверждения от Telegram
4. После подтверждения вы автоматически войдете в админ-панель (если ваш аккаунт в вайт-листе)

## Проверка настроек

Если проблема сохраняется, проверьте:

- ✅ Домен настроен в BotFather через `/setdomain`
- ✅ Сайт доступен по HTTPS
- ✅ Бот существует и активен
- ✅ Ваш аккаунт добавлен в вайт-лист администраторов (`npm run seed:admin-telegram`)
